# DA사업부 업무 노드 시뮬레이터

삼성전자 DA사업부의 업무 프로세스를 노드-엣지 그래프로 시각화하고,
노드(업무 단위)를 제거했을 때 절감되는 비용을 실시간으로 계산하는 인터랙티브 웹앱.

## 기능
- 142개 노드 / 240+ 엣지로 구성된 풀 라이프사이클 (소비자 인사이트 → POD → 개발 → 시험 → 양산준비 → 본양산 → 물류·영업 → 서비스 → 분석 → 백오피스 → 사이클 클로즈)
- 노드 클릭 제거 시 절감 시간/비용/FTE 실시간 카운트업
- 부서/직군/엣지 타입 필터, AX 후보 토글
- 노드/엣지 추가·편집·삭제·복원, 결과 클립보드 복사
- 화살표 길이가 병목 크기(연간 처리 시간)에 비례

## 사용
- 단일 HTML 파일 — `node_simulator.html` 더블클릭 또는 호스팅 URL로 접속
- 외부 의존: vis-network (CDN), Nanum Gothic (Google Fonts)
- 시간당 비용 기준: 60,000원/h, FTE 기준: 2,080h/년
